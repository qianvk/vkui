# Third-party notices

## Fira Code Nerd Font

The Gallery embeds `FiraCodeNerdFont-Regular.ttf` from the official Nerd Fonts
release archive. It is derived from Fira Code and includes glyphs from the Nerd
Fonts aggregation project.

The font is used only by the Gallery executable; it is not linked into
`VkUI::Core` or `VkUI::Widgets`, and it is not installed into the operating
system. The upstream license and release notes are preserved in
`resources/fonts/`.

- Nerd Fonts: <https://github.com/ryanoasis/nerd-fonts>
- Fira Code: <https://github.com/tonsky/FiraCode>
